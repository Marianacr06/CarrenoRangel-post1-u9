Área protegida

Bienvenido

Acceso autorizado con rol ROLE.

Autenticación

Login basado en formulario con `UserDetailsService`, persistencia MySQL y contraseñas cifradas con BCrypt.

Autorización

Las rutas `/admin/**` requieren ADMIN y `/user/**` aceptan USER y ADMIN.

Registro

Los usuarios nuevos se registran con el rol USER y credenciales validadas en servidor.
